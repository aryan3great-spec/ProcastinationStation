Hello hello hello , this is the start to my hopefully long line of emulators that i hope to perfect to my comfort. 
I love Pokemon hence im here to make a emulator thats gameboy shaped, it is powered by the raspberry pi and i have made 
a custom enclosure for it as well 

I got the inspiration from multiple videos on youtube and also from my friend buying a used new 3ds (terrible naming) and that got me so interested in emulators. From the videos ik that the most cost effiecent and easy to start with sbc is going to be the rpi 02w. 

The only function it has is to run retropie and as many emulators i can fit on a sd card

It has a dpad, abxy, start select and 4 shoulder buttons which should be able to cover emsot games. Which i hope look greaat on the 2.8 inch non touch lcd i've chosen to save cost 

I had some troubles in kicad with locating the buttons equidistant and allign and im certain they arent even symmetric but we'll know soon enough 

i hope to add audio and power in further itterations but idek if i can make the screen work yet so thats why its not as complicated
